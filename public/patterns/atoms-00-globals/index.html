Icons

▼

Icons are provided using the Font Awesome icon font, currently version 4.6.3. For a complete list of icons and classes associated with each, visit Font Awesome's website. Also be sure to check out examples for how you can modify icon behavior and presentation for a variety of use cases.

It is imperative that you apply appropriate markup for accessibility support. Refer to Font Awesome's accessibility guide for more details.
