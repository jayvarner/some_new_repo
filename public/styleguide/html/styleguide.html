inline elements complete

▼

<a> – text link

<em> – Adds semantic emphasis

<strong> – Used to indicate semantically greater importance than just emphasis

<i> – Used to for purely stylistic purposes with no semantic emphasis

<b> – Used for stylistic purposes with no semantic implication of importance

<u> – Just don't use this. Ever.

<del> – Used to communicate deleted text.

<ins> – Used to indicate text has been inserted.

<s> – This text has a strikethrough.

<sup> – Superscript® is used for, well, superscripts.

<sub> – Subscript is used for things like H2O

<small> – This small text is small for for fine print, etc.

<abbr> – Abbreviation – HTML

<kbd> – Used for keyboard input such as Cmd

<q> – This text is a short inline quotation

<cite> – Used for referencing a source such as a book title

<dfn> – The dfn element is indicates a definition.

<mark> – The mark element indicates a highlight

<code> – This is what inline code looks like.

<samp> – This is sample output from a computer program: your site has been generated...

<var> – The variarble element, such as x = y

Since 2002 the Stuart A.
